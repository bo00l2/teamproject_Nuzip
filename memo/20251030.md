주제 : 뉴스 요약 & 감정 분석 대시보드

** 기능
- 뉴스 기사 AI 요약 -> Gemini
- 뉴스 감정 분석
- AI 요약 시스템 피드백 별점 or 좋/싫 클릭
- 관심있는 기사 스크랩 (카테고리별로 정리할 수 있게)
- 관심있는 기사에 대해 메모할 수 있는 메모장 기능 (window 스티커 메모같은!)
- 날짜 / 시간별로 기사 업데이트
- 감정 비율 차트 시각화
- 시간대별 감정 변화(감정 변화 추세 확인) 
- 뉴스 카테고리를 클릭했을 때, 카테고리 관련 키워드 띄워주기
- 관심 카테고리의 기사가 업데이트 됐을 때 알림 설정 기능

회원가입 시스템
- 회원가입 -> 로그인 -> 카테고리 선정 -> 첫화면에 선정한 카테고리 위주의 알고리즘 설계

카테고리 페이지에 관련 카테고리 기사들 띄우고,
밑에 가장 많이 검색된 키워드 띄움
한 페이지당 한 카테고리 

* 사용자 모드
- 관심 키워드를 클릭했을 때 요약된 기사 볼 수 있음
- AI 요약 시스템 평가 별점 or 좋/싫 클릭
- 관심있는 기사 스크랩 (카테고리별로 정리할 수 있게)
- 관심있는 기사에 대해 메모할 수 있는 메모장 기능 (window 스티커 메모같은!)


* 관리자 모드
1. 데이터 관리 기능
- 뉴스 데이터 수집
- 뉴스 데이터 검수
- 요약 / 감정 재처리
- 데이터 일괄 삭제

2. 사용자 및 접근 관리
- 관리자 계정 관리
- 일반 사용자 접근 제어
- 활동 로그 확인

3. 시각화 & 통계 관리
- 대시보드 통계 모니터링
- KPI 확인 (수집 기사 수, 요약 처리 완료율, API 사용량 등)
- 시각화 설정

4. 시스템 / API 관리
- API 호출 관리
- 모델 관리 
- DB 백업/복구


우선 작업
- 회원가입 -> 로그인 -> 카테고리 선정 -> 첫화면에 선정한 카테고리 위주의 알고리즘 설계
- 로그인 / 로그아웃 창


로그인 전
- 가나다순으로 기사 정렬


로그인 후
- 상단에 관심분야 


Gemini API 가져오기

## 금일 일정
- Gemini API 가져오기

http://localhost:8080/gemini?prompt={}
{}에 질문 작성하면 gemini 답변이 나오도록 성공함.

SpringBoot 실행하기 전에 GEMINI_API_KEY 를 Google AI Studio에 로그인해서 개인 키를 가져와야함.
